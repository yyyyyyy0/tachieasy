tachieasy / たちいーじー の使い方

TRPG セッション用の立ち絵を一括バストアップトリミング → タイル合成する単一 HTML ツール。すべての処理はブラウザ内で完結し、画像は外部に送信されません。

5 ステップで完了

  1. 画像を選ぶ: 「ファイル選択」で複数枚を選択。サムネをクリックで参照画像に切替。
  2. ROI を決める: 「アス比」プリセット (1:1 / 3:4 / 4:5 / 2:3 / 16:9) を選び、必要なら「ロック」をオン。矩形をドラッグ・隅をリサイズ。「バストアップ プリセット」で目安。
  3. 一括トリミング: 「この範囲で全画像をトリミング」でまとめて切り抜き。比率モードなら異サイズ OK。
  4. 並び替え: トリミング結果のサムネをドラッグで並び替え。
  5. タイリング & DL: 列数・背景・形式を設定 → 「プレビュー生成」 → 「タイル画像を DL」。

背景色

  • 透過: PNG のみ。チェックを外すと色を指定可。
  • ColorPicker / hex 入力: 双方向同期。#RRGGBB で入力。
  • JPEG モード: 透過不可。自動で透過が OFF になります。

アス比ロックの挙動

  • プリセットを押すと自動でロック ON。中心を保ったまま指定比率に整形。
  • ロック中はドラッグ・数値入力どちらも比率を維持。「自由」を選ぶとロック解除。

テーマ

  • 右上「テーマ」ボタンで auto → light → dark を循環。設定はブラウザに保存。

あなたの絵は外に出ません

  • このページに渡した立ち絵は、あなたのブラウザの中だけで処理されます。どこかのサーバに送られたり、保存されたりすることはありません。
  • Wi-Fi を切ってもすべての機能が動きます。試しに機内モードでも使えます。
  • 誰かに見られたり、AI 学習に勝手に使われたりすることはありません。安心してオリジナルキャラの立ち絵を入れてください。
  • 使ったあとはタブを閉じれば、画像はあなたの端末からも消えます (DL したものは残ります)。
